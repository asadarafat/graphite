# SSH to network nodes from a browser

## Pre-requisites

1. Install [Node Version Manager](https://github.com/nvm-sh/nvm), Node.js and npm

```Shell
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
source ~/.bashrc
nvm install 16
node -v
npm -v
````
